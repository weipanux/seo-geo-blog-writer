## Pre-Publish SEO Checklist

- [ ] Primary keyword in title (within first 60 characters)
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in URL slug
- [ ] Primary keyword in meta description
- [ ] 2-3 H2 headings with keyword variations
- [ ] 4-8 H2 headings total with clear hierarchy
- [ ] Meta title: 50-60 characters
- [ ] Meta description: 145-155 characters with CTA
- [ ] 3-5 internal links with descriptive anchor text
- [ ] 2-4 external authority links to quality sources
- [ ] 5-8 images with optimized alt text
- [ ] Images compressed (<100KB each)
- [ ] FAQ section included (4-8 questions minimum)
- [ ] FAQ schema markup prepared
- [ ] Author bio with credentials and photo
- [ ] Publication and last updated dates included
- [ ] Article schema markup prepared
- [ ] Table of contents with jump links (for 1,500+ words)
- [ ] Mobile-friendly formatting verified
- [ ] Readability score: 60-70 (Flesch Reading Ease)
- [ ] Short paragraphs (2-4 sentences average)
- [ ] Varied sentence structure and length
- [ ] Active voice used in 80%+ of sentences
- [ ] Disclosure statement if applicable
- [ ] All facts verified and sources cited
- [ ] All links tested (no 404s)
- [ ] Spell check and grammar check completed
- [ ] Preview on mobile device
- [ ] Social sharing preview checked
