# SEO Checklist

## Pre-Writing: Keyword Research

### Primary Keyword Selection
- [ ] Identified main target keyword
- [ ] Verified search volume (minimum 100 searches/month for viability)
- [ ] Analyzed keyword difficulty (target KD <30 for new sites, <50 for established)
- [ ] Confirmed search intent matches content type
- [ ] Checked if keyword is achievable based on domain authority

### Secondary Keywords
- [ ] Found 3-5 related secondary keywords
- [ ] Identified long-tail variations (3-5 words)
- [ ] Listed LSI (Latent Semantic Indexing) keywords
- [ ] Mapped keywords to specific sections of content

### Competitor Analysis
- [ ] Analyzed top 3-5 ranking articles
- [ ] Identified common content patterns
- [ ] Found content gaps to exploit
- [ ] Noted average word count of top results
- [ ] Identified common sub-topics covered

## Content Structure

### Title Optimization
- [ ] Primary keyword in title
- [ ] Title is 50-60 characters (optimal for SERPs)
- [ ] Title is compelling and click-worthy
- [ ] Includes numbers, power words, or emotional triggers
- [ ] Avoid ALL CAPS or excessive punctuation

### URL Slug
- [ ] Primary keyword in URL slug
- [ ] Short and descriptive (3-5 words max)
- [ ] Uses hyphens (not underscores)
- [ ] Lowercase only
- [ ] No special characters or stop words

### Meta Description
- [ ] Primary keyword included
- [ ] 145-155 characters (optimal for display)
- [ ] Compelling call-to-action
- [ ] Accurately describes content
- [ ] Includes benefit or value proposition

### Heading Structure
- [ ] One H1 tag only (typically the title)
- [ ] Primary keyword in H1
- [ ] 4-8 H2 headings (main sections)
- [ ] 2-3 H2s include keyword variations
- [ ] H3s used for sub-sections under H2s
- [ ] Clear hierarchy (no skipping levels)
- [ ] Descriptive headings (not vague)

## Content Quality

### Introduction (First 100-150 Words)
- [ ] Primary keyword in first 100 words
- [ ] Hook that captures attention
- [ ] Clear statement of what article covers
- [ ] Addresses reader's pain point or question
- [ ] Sets expectations for content length/depth

### Body Content
- [ ] Minimum word count met (1,000+ for basic, 2,000+ for competitive)
- [ ] Primary keyword density: 1-2%
- [ ] Secondary keywords naturally distributed
- [ ] LSI keywords included throughout
- [ ] Short paragraphs (2-4 sentences)
- [ ] Mix of paragraph lengths for rhythm
- [ ] Transition words between sections
- [ ] Active voice used (80%+ of sentences)

### Readability
- [ ] Flesch Reading Ease: 60-70
- [ ] Grade level: 8th grade or lower (for general audience)
- [ ] Short sentences (average 15-20 words)
- [ ] Simple vocabulary (avoid jargon unless technical audience)
- [ ] Varied sentence structure

### Content Depth
- [ ] Topic thoroughly covered
- [ ] All major sub-topics addressed
- [ ] Answers common questions
- [ ] Provides actionable information
- [ ] Includes examples or case studies
- [ ] Original insights or perspectives

## Internal Linking

### Strategic Links
- [ ] 3-5 internal links included
- [ ] Links to related blog posts
- [ ] Links to pillar content/category pages
- [ ] Links to relevant product/service pages
- [ ] Anchor text is descriptive (not "click here")
- [ ] Links open in same tab
- [ ] Links to pages you want to rank

### Link Distribution
- [ ] Links naturally distributed throughout content
- [ ] No more than 1 link per 300-500 words
- [ ] Links add value to reader experience
- [ ] Mix of exact match and partial match anchor text

## External Linking

### Authority Links
- [ ] 2-4 external links to authoritative sources
- [ ] Links support claims with data/research
- [ ] Sources are recent (within 2-3 years)
- [ ] Links go to .edu, .gov, or reputable industry sites
- [ ] Avoid linking to competitors
- [ ] External links open in new tab (optional)

### Citation Best Practices
- [ ] Sources clearly attributed (e.g., "According to...")
- [ ] Statistics include date and source
- [ ] Expert quotes properly credited
- [ ] Links verify to original source (not aggregators)

## Visual Elements

### Images
- [ ] 5-8 images minimum
- [ ] Featured image at top (1200x630px for social sharing)
- [ ] Images relevant to content
- [ ] High quality but compressed (<100KB each)
- [ ] All images have descriptive alt text
- [ ] Alt text includes keyword (when natural)
- [ ] File names are descriptive (not IMG_1234.jpg)
- [ ] Images are properly sized (not oversized)

### Other Visual Elements
- [ ] Tables for data comparison (when appropriate)
- [ ] Bullet points for easy scanning
- [ ] Numbered lists for step-by-step processes
- [ ] Screenshots or examples (for how-to content)
- [ ] Infographics or charts (for data-heavy topics)
- [ ] Video embeds (if available)

## User Engagement

### Call-to-Action (CTA)
- [ ] Clear CTA in conclusion
- [ ] CTA aligned with content goal
- [ ] Easy next step defined
- [ ] Link/button included (if applicable)

### FAQ Section
- [ ] 4-8 frequently asked questions
- [ ] Questions match "People Also Ask" queries
- [ ] Questions use natural language
- [ ] Answers are 40-60 words
- [ ] Questions include keyword variations
- [ ] FAQ schema markup prepared

### Table of Contents
- [ ] TOC for articles 1,500+ words
- [ ] Jump links to main sections
- [ ] Clean, scannable format
- [ ] Positioned after introduction

## E-E-A-T Signals

### Author Credibility
- [ ] Author name included
- [ ] Author bio at end (100-150 words)
- [ ] Author credentials/expertise mentioned
- [ ] Author photo included
- [ ] Links to author's social profiles
- [ ] Author's relevant experience highlighted

### Content Trustworthiness
- [ ] Publication date displayed
- [ ] Last updated date shown (if updated)
- [ ] Facts are accurate and verifiable
- [ ] Sources are authoritative
- [ ] No false or misleading claims
- [ ] Disclosures added (affiliate links, sponsorships)

### Expertise Demonstration
- [ ] First-hand experience mentioned (when applicable)
- [ ] Original research or testing included
- [ ] Industry-specific terminology used correctly
- [ ] Depth shows true expertise
- [ ] Multiple perspectives considered

## Technical SEO

### Schema Markup
- [ ] Article schema added
- [ ] FAQ schema added (if FAQ section exists)
- [ ] Author schema included
- [ ] Organization schema (for business site)
- [ ] Breadcrumb schema (if applicable)
- [ ] Review schema (for product reviews)

### Page Speed
- [ ] Images compressed
- [ ] No unnecessary scripts/plugins
- [ ] Mobile-optimized
- [ ] Core Web Vitals considered

### Mobile Optimization
- [ ] Responsive design verified
- [ ] Text readable without zooming
- [ ] Tap targets appropriately sized
- [ ] No horizontal scrolling
- [ ] Fast mobile load time

## Pre-Publish Checklist

### Final Review
- [ ] Spell check completed
- [ ] Grammar check completed
- [ ] All links tested (no 404s)
- [ ] Images load properly
- [ ] Formatting consistent
- [ ] No Lorem Ipsum or placeholder text
- [ ] Preview on mobile device

### SEO Tools Check
- [ ] Run through Yoast/RankMath (if using)
- [ ] Check with Hemingway Editor (readability)
- [ ] Verify with SEO extension (keyword usage)
- [ ] Test meta preview in SERP simulator

### Social Sharing
- [ ] Open Graph tags added
- [ ] Twitter Card tags added
- [ ] Featured image optimized for social
- [ ] Preview how it appears when shared

## Post-Publish

### Indexing
- [ ] Submit URL to Google Search Console
- [ ] Request indexing
- [ ] Share on social media for crawling signals

### Monitoring
- [ ] Set up tracking in analytics
- [ ] Monitor rankings for target keywords
- [ ] Track click-through rate in Search Console
- [ ] Review performance after 2-4 weeks

### Updates
- [ ] Plan content refresh (6-12 months)
- [ ] Update statistics and dates
- [ ] Add new internal links from new content
- [ ] Refresh images if needed

## Keyword Density Guidelines

**Primary Keyword:**
- Title: 1 time
- H1: 1 time
- H2s: 2-3 times (natural variations)
- First 100 words: 1 time
- Body: 1-2% density (5-8 times in 1,000 words)
- Meta description: 1 time
- URL: 1 time
- Image alt text: 1-2 times

**Avoid Keyword Stuffing:**
- If it feels unnatural, it is
- Use synonyms and variations
- Prioritize readability over density
- Context matters more than exact matches

## Content Types & Word Count Targets

- **Definitive Guide**: 3,000-5,000 words
- **How-To Article**: 1,500-2,500 words
- **Listicle**: 1,000-2,000 words
- **Product Review**: 1,500-2,000 words
- **Comparison Post**: 2,000-3,000 words
- **Case Study**: 1,500-2,500 words
- **News/Opinion**: 800-1,500 words

*Note: Always match or exceed top-ranking content length for competitive keywords.*
